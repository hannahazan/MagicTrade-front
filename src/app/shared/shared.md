## `shared/` - UI réutilisable partout (pas métier)

__Contient :__
- Composants UI génériques (`ButtonComponent`, `ModalComponent`...)
- Pipes réutilisables (exemples : `CapitalizePipe`, `DateFormatPipe`)
- Directives génériques

```
shared/
├── components/
│   ├── button/
│   └── modal/
├── directives/
└── pipes/
```
